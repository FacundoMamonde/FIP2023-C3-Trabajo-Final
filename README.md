# SOAT

## How to setup server

### In console type:
```
git clone https://github.com/FacundoMamonde/SOAT-FRONT
```
### Switch to folder
```
cd FIP2023-C3-Trabajo-Final
```
### Installing libreries
```
npm install
```
### Compiles and hot-reloads for development
```
npm run serve
```
